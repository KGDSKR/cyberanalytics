# CyberAnalytics

Telegram Mini App: ИИ-аналитика матчей CS2 на базе Claude + PandaScore.

## Быстрый старт

```bash
npm install
copy .env.example .env   # и вписать ключи
npm run dev              # http://localhost:5173
```

Без ключей приложение работает в демо-режиме на тестовых данных.

Подробности — в [CLAUDE.md](CLAUDE.md).
